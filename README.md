### Requerimientos
- Node >= 16.13.1
- npm >= 8.1.2

### Para iniciar la página

Instalar dependencias

```
npm install
```

Iniciar

```
npm run dev
```


### Paquetes incluídos
- [React Router](https://reactrouter.com/en/main)
- [Material UI](https://mui.com/material-ui/getting-started/usage/)
- [Bootstrap](https://getbootstrap.com/)
